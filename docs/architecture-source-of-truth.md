# Architecture Source of Truth

The canonical architecture for the SARATHI platform is [Platform_Architecture.md](../Platform_Architecture.md).

## Why This Exists
The earlier architecture documents were written when the platform was still being described too narrowly as a mobile-first solution. This note points contributors to the corrected platform-first architecture that includes:
- Citizen Mobile App
- Citizen Web Portal
- Government Administrative Portal
- Shared backend services
- Shared AI/OCR services
- Shared workflow engine
- Shared integration layer

## Rule
When there is any conflict between older architecture notes and [Platform_Architecture.md](../Platform_Architecture.md), the platform architecture document wins.
